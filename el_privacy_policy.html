Καλώς ήρθατε στο Defanzo. Εκτιμούμε την εμπιστοσύνη σας και δεσμευόμαστε να προστατεύσουμε το απόρρητό σας. Αυτή η πολιτική στοχεύει στο να διευκρινίσει τις πρακτικές μας σχετικά με τη συλλογή, χρήση και προστασία των πληροφοριών. Παρακαλούμε διαβάστε προσεκτικά αυτήν την πολιτική για να κατανοήσετε πώς χειριζόμαστε τα δεδομένα σας.

1. Επισκόπηση της Εφαρμογής

Το Defanzo είναι μια εφαρμογή σχεδιασμένη να καλύψει τις ανάγκες σας στην επεξεργασία ψηφιακών μέσων. Η εφαρμογή προσφέρει μια σειρά υπηρεσιών, συμπεριλαμβανομένων:

  • Συμπίεση εικόνων και βίντεο
  • Αλλαγή μεγέθους εικόνων
  • Μετατροπή μορφών εικόνας
  • Περικοπή βίντεο

Ο στόχος μας είναι να παρέχουμε ένα αποτελεσματικό και εύχρηστο εργαλείο για τη βελτιστοποίηση και διαχείριση των αρχείων πολυμέσων σας, διατηρώντας παράλληλα τα υψηλότερα πρότυπα απορρήτου και ασφάλειας.

2. Συλλογή και Χρήση Πληροφοριών

2.1 Firebase Crashlytics

Χρησιμοποιούμε το Firebase Crashlytics για τη βελτίωση της σταθερότητας και της απόδοσης της εφαρμογής. Αυτή η υπηρεσία συλλέγει:

  • Πληροφορίες σχετικά με σφάλματα και κρασαρίσματα που μπορεί να συμβούν κατά τη χρήση της εφαρμογής
  • Τύπο συσκευής και έκδοση λειτουργικού συστήματος
  • Έκδοση της εφαρμογής που χρησιμοποιείται

Ο σκοπός της συλλογής αυτών των δεδομένων είναι να μας επιτρέψει να εντοπίσουμε και να διορθώσουμε γρήγορα προβλήματα, εξασφαλίζοντάς σας μια ομαλή και απρόσκοπτη εμπειρία χρήσης.

2.2 Firebase Analytics

Χρησιμοποιούμε το Firebase Analytics για να κατανοήσουμε πώς οι χρήστες χρησιμοποιούν την εφαρμογή μας. Συλλέγουμε ανώνυμα δεδομένα σχετικά με:

  • Αριθμό ανοιγμάτων της εφαρμογής
  • Πιο συχνά χρησιμοποιούμενες λειτουργίες
  • Γενικά μοτίβα χρήσης

Αυτές οι πληροφορίες μας βοηθούν να λαμβάνουμε ενημερωμένες αποφάσεις για τη βελτίωση της εφαρμογής και την ανάπτυξη νέων λειτουργιών που ανταποκρίνονται καλύτερα στις ανάγκες σας.

2.3 Firebase Messenger

Χρησιμοποιούμε το Firebase Messenger για να σας στέλνουμε σημαντικές ειδοποιήσεις. Αυτές οι ειδοποιήσεις μπορεί να περιλαμβάνουν:

  • Νέες ενημερώσεις για την εφαρμογή
  • Ανακοινώσεις για νέες λειτουργίες
  • Σημαντικές πληροφορίες σχετικά με την ασφάλεια και την απόδοση της εφαρμογής

Στόχος μας είναι να σας κρατάμε ενήμερους για τις τελευταίες εξελίξεις και να διασφαλίζουμε ότι έχετε την καλύτερη δυνατή εμπειρία.

2.4 Google AdMob

Χρησιμοποιούμε το Google AdMob για την προβολή διαφημίσεων εντός της εφαρμογής. Μέσω αυτής της υπηρεσίας:

  • Το AdMob μπορεί να συλλέγει πληροφορίες σχετικά με τη συσκευή σας και την αλληλεπίδρασή σας με τις διαφημίσεις
  • Αυτές οι πληροφορίες χρησιμοποιούνται για την παροχή διαφημίσεων πιο σχετικών με τα ενδιαφέροντά σας
  • Δεν μοιραζόμαστε προσωπικές πληροφορίες με διαφημιστές

Τηρούμε τις πολιτικές απορρήτου του Google AdMob και εργαζόμαστε για να διασφαλίσουμε ότι το απόρρητό σας δεν παραβιάζεται.

3. Απαιτούμενες Άδειες

3.1 Πρόσβαση στον Αποθηκευτικό Χώρο

Ζητάμε άδεια πρόσβασης στον αποθηκευτικό χώρο της συσκευής σας για τους εξής λόγους:

  • Ανάγνωση των εικόνων και βίντεο που επιθυμείτε να επεξεργαστείτε
  • Αποθήκευση των επεξεργασμένων αρχείων (συμπιεσμένων, μετατρεπόμενων, περικομμένων) στη συσκευή σας

Χωρίς αυτήν την άδεια, η εφαρμογή δεν θα μπορούσε να εκτελέσει τις βασικές λειτουργίες της. Σας διαβεβαιώνουμε ότι δεν έχουμε πρόσβαση σε άλλα αρχεία εκτός από αυτά που επιλέγετε εσείς οι ίδιοι για επεξεργασία.

3.2 Άδεια Ειδοποιήσεων

Ζητάμε άδεια για την αποστολή ειδοποιήσεων για τη βελτίωση της εμπειρίας χρήσης της εφαρμογής:

  • Να σας ενημερώνουμε για την πρόοδο μακροχρόνιων διαδικασιών επεξεργασίας, όπως η συμπίεση μεγάλων αρχείων
  • Να σας ειδοποιούμε για σημαντικές ενημερώσεις της εφαρμογής
  • Να στέλνουμε συμβουλές για τη βελτίωση της χρήσης της εφαρμογής

Μπορείτε πάντα να απενεργοποιήσετε τις ειδοποιήσεις από τις ρυθμίσεις της συσκευής σας αν δεν επιθυμείτε να τις λαμβάνετε.

4. Προστασία του Απορρήτου σας

Δεσμευόμαστε να προστατεύουμε το απόρρητό σας στον μέγιστο δυνατό βαθμό. Για να το επιτύχουμε αυτό:

  • Καθώς δεν συλλέγουμε αναγνωρίσιμα προσωπικά δεδομένα, δεν έχουμε δεδομένα που θα μπορούσαμε να σας παρέχουμε ατομικά.
  • Δεν απαιτούμε δημιουργία λογαριασμού ή σύνδεση για τη χρήση της εφαρμογής
  • Τα δεδομένα που συλλέγουμε είναι τεχνικής και μη προσωπικής φύσης, επομένως δεν μπορούν να τροποποιηθούν ή να διορθωθούν.

5. Τα Δικαιώματα και οι Επιλογές σας

Σεβόμαστε το δικαίωμά σας να ελέγχετε τα δεδομένα σας. Έχετε το δικαίωμα να:

  • Ζητήσετε πληροφορίες σχετικά με τα δεδομένα που συλλέγουμε για εσάς
  • Ζητήσετε τη διαγραφή οποιωνδήποτε δεδομένων σχετίζονται με εσάς (λαμβάνοντας υπόψη ότι δεν συλλέγουμε προσωπικά δεδομένα)
  • Εάν επιθυμείτε να σταματήσετε να μοιράζεστε δεδομένα κρασαρίσματος, μπορείτε να το κάνετε απεγκαθιστώντας την εφαρμογή ή σταματώντας να τη χρησιμοποιείτε.

6. Συμμόρφωση με τους Νόμους και Προστασία Δεδομένων

Σεβόμαστε το απόρρητο των χρηστών μας και δεσμευόμαστε να συμμορφωνόμαστε με τους ισχύοντες νόμους, συμπεριλαμβανομένου του Γενικού Κανονισμού Προστασίας Δεδομένων (GDPR) όπου είναι σχετικό:

  • Περιορισμένη συλλογή δεδομένων: Συλλέγουμε μόνο ανώνυμα δεδομένα κρασαρίσματος για τη βελτίωση της απόδοσης της εφαρμογής. Δεν συλλέγουμε προσωπικά αναγνωρίσιμες πληροφορίες.
  • Χρήση του Firebase Analytics και AdMob: Αυτές οι υπηρεσίες μπορεί να συλλέγουν ανώνυμα δεδομένα σχετικά με τη χρήση της εφαρμογής για τη βελτίωση της απόδοσης και την εμφάνιση σχετικών διαφημίσεων.
  • Δικαιώματα χρηστών: Σεβόμαστε το δικαίωμα των χρηστών μας στο απόρρητο. Εάν έχετε ερωτήσεις ή ανησυχίες σχετικά με τα δεδομένα σας, παρακαλούμε επικοινωνήστε μαζί μας.
  • Ασφάλεια: Λαμβάνουμε κατάλληλα μέτρα ασφαλείας για την προστασία των συλλεγόμενων δεδομένων από μη εξουσιοδοτημένη πρόσβαση, τροποποίηση ή αποκάλυψη.

7. Αλλαγές στην Πολιτική Απορρήτου

Ενδέχεται να ενημερώνουμε αυτήν την πολιτική κατά καιρούς για να αντικατοπτρίζει αλλαγές στις πρακτικές μας ή για λειτουργικούς, νομικούς ή κανονιστικούς λόγους. Σε περίπτωση σημαντικών αλλαγών:

  • Θα σας ενημερώσουμε μέσω ειδοποίησης εντός της εφαρμογής
  • Θα ζητήσουμε τη συγκατάθεσή σας για τους νέους όρους πριν συνεχίσετε να χρησιμοποιείτε την εφαρμογή
  • Θα παρέχουμε τη δυνατότητα επανεξέτασης της προηγούμενης έκδοσης της πολιτικής
